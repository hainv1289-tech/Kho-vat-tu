Sao lưu dữ liệu

Dữ liệu (mã, ảnh, ghi chú) đang lưu trong Safari trên máy này. Nếu xoá lịch sử Safari hoặc đổi máy, dữ liệu sẽ mất. Hãy xuất file sao lưu định kỳ để không mất công nhập lại.

📤 Xuất dữ liệu
Lưu toàn bộ vật tư + ảnh thành 1 file. Lưu file này vào iCloud Drive hoặc Files để an toàn.
📥 Nhập lại dữ liệu
Khôi phục từ file đã xuất trước đó. Sẽ thay thế toàn bộ dữ liệu hiện tại.
